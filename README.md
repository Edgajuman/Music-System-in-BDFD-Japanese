
# BDFD v7.0 ベータ版以降向け音楽システム (非公開) 日本語版

Bot Designer For Discord でDiscordボットと音楽を再生するための高度なシステムです。
**本システムはリソース節約、ユーザーフレンドリー化、新BDFD制限への対応を目的に再設計されました**


## 設定

#### 必須変数は1つだけ作成すれば十分です


| 名前 | 値     |
| :-------- | :------- |
| `TOKEN` | `Discordボットのトークン` |

## コマンド
- ! (このコードは6つのコマンドを1つにまとめたものなので、プレフィックスのみ入力してください。例: !)
- !遊ぶ
- $onInteraction[Musics]

## 機能内容
> 現時点では以下の機能のみ:
- 音楽再生
- 一時停止・再開システム
- 早送り・巻き戻しシステム
- キュー内の任意の楽曲へジャンプするシステム（最大25曲）
- 音楽停止とボット切断機能

## 想定される不具合
システムは完全に機能しますが、後ほど修正されるいくつかの不具合があります。
主な不具合は以下の通りです：
- 音楽の処理に時間がかかる場合があります（主に検索対象の動画/音楽のサイズと再生時間による）
- 音楽キューが終了してもループ再生されない場合があります（今後のアップデートで修正予定）
